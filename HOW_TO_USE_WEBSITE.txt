═══════════════════════════════════════════════════════════════════
  🚀 HOW TO USE YOUR MIT ADT SMART CAMPUS PORTAL
═══════════════════════════════════════════════════════════════════

IMPORTANT: The server must be RUNNING for the website to work!

═══════════════════════════════════════════════════════════════════
  📋 EVERY TIME YOU WANT TO USE THE WEBSITE
═══════════════════════════════════════════════════════════════════

STEP 1: Start the Server
   
   Option A (EASIEST):
   → Double-click: START_SERVER.bat
   
   Option B (Manual):
   → Open PowerShell in project folder
   → Type: npm start
   → Press Enter

STEP 2: Wait for Server to Start
   
   You'll see these messages:
   ✓ "MIT ADT Smart Campus Portal running on port 3000"
   ✓ "Visit: http://localhost:3000"
   ✓ "✅ Connected to MongoDB successfully!"
   
   This takes 3-5 seconds

STEP 3: Open Your Browser
   
   → Open Chrome, Firefox, Edge, or any browser
   → Go to: http://localhost:3000
   → Website will load!

STEP 4: Use the Website
   
   → Login with test accounts
   → Browse all features
   → Add data, manage portal

STEP 5: When Done
   
   → Go back to the server window
   → Press Ctrl + C to stop server
   → Or just close the window

═══════════════════════════════════════════════════════════════════
  ⚠️  IMPORTANT RULES
═══════════════════════════════════════════════════════════════════

✓ Server window must stay OPEN while using website
✓ Don't close the black/blue terminal window
✓ If you close it, website stops working
✓ To use website again, restart server (double-click START_SERVER.bat)

═══════════════════════════════════════════════════════════════════
  ❓ WHY DO I NEED TO START THE SERVER?
═══════════════════════════════════════════════════════════════════

This is a LOCAL web application that runs on YOUR computer.

Think of it like:
- Microsoft Word: You open it when you need it
- MIT ADT Portal: You start server when you need it

The server is like "opening" the application.
When server runs = Website works
When server stops = Website doesn't work

This is NORMAL for web applications running locally!

  💡 TIPS & TRICKS

TIP 1: Create Desktop Shortcut
   → Right-click START_SERVER.bat
   → Send to → Desktop (create shortcut)
   → Now you can start from desktop!

TIP 2: Keep Server Window Minimized
   → After starting, minimize the window
   → It runs in background
   → Website keeps working

TIP 3: Check if Server is Running
   → Look for the black/blue window
   → If you see it = Server is running
   → If not = Need to start server

TIP 4: Bookmark the Website
   → In browser, bookmark: http://localhost:3000
   → Quick access when server is running

═══════════════════════════════════════════════════════════════════
  🐛 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════

Problem: "localhost refused to connect"
Solution: Server is not running
   → Double-click START_SERVER.bat
   → Wait 5 seconds
   → Try again

Problem: "Port 3000 already in use"
Solution: Server is already running somewhere
   → Check for open terminal windows
   → Close them and try again
   → Or restart computer

Problem: Server window closes immediately
Solution: There's an error
   → Open PowerShell manually
   → Type: npm start
   → Read the error message
   → Usually MongoDB connection issue

Problem: Website loads but can't login
Solution: MongoDB not connected
   → Check server window for errors
   → Should see "✅ Connected to MongoDB"
   → If not, check MongoDB setup
