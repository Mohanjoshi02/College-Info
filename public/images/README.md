# Campus Images

This directory contains campus images for the MIT ADT Smart Campus Portal.

## Required Images:
- campus-1.jpg - Main campus building
- campus-2.jpg - Central dome
- campus-3.jpg - Campus architecture
- campus-4.jpg - Aerial view
- campus-5.jpg - Campus facilities
- campus-6.jpg - University grounds

## Note:
The images provided in the chat have been saved and should be placed in this directory with the corresponding filenames above.

## Image Requirements:
- Format: JPG/JPEG
- Recommended size: 800x600 pixels or higher
- File size: Under 2MB each for optimal loading
