  📅 MIT ADT PORTAL - DAILY USE GUIDE

This guide explains how to use your portal every day.

═══════════════════════════════════════════════════════════════════
  🚀 STARTING THE PORTAL (EVERY TIME)
═══════════════════════════════════════════════════════════════════

METHOD 1: Using Batch File (EASIEST)
   
   1. Go to project folder:
      C:\Users\mohan\OneDrive\Desktop\dt prototype 2
   
   2. Double-click: START_SERVER.bat
   
   3. A window will open showing:
      "MIT ADT Smart Campus Portal running on port 3000"
   
   4. Keep this window OPEN (minimize it if you want)
   
   5. Open browser and go to: http://localhost:3000

METHOD 2: Using Desktop Shortcut (RECOMMENDED)
   
   1. First time only: Double-click CREATE_DESKTOP_SHORTCUT.bat
   
   2. A shortcut "MIT ADT Portal" will appear on desktop
   
   3. From now on, just double-click desktop shortcut!
   
   4. Server starts automatically
   
   5. Open browser: http://localhost:3000

METHOD 3: Using PowerShell (MANUAL)
   
   1. Open PowerShell in project folder
   
   2. Type: npm start
   
   3. Press Enter
   
   4. Wait for "running on port 3000" message
   
   5. Open browser: http://localhost:3000

═══════════════════════════════════════════════════════════════════
  🛑 STOPPING THE PORTAL (WHEN DONE)
═══════════════════════════════════════════════════════════════════

OPTION 1: Close Server Window
   → Simply close the black/blue terminal window
   → Server stops immediately

OPTION 2: Press Ctrl + C
   → Go to server window
   → Press Ctrl + C
   → Type 'Y' if asked
   → Server stops gracefully

OPTION 3: Just Leave It Running
   → If you'll use it again soon
   → Keep server window minimized
   → Website stays accessible

═══════════════════════════════════════════════════════════════════
  ❓ COMMON QUESTIONS
═══════════════════════════════════════════════════════════════════

Q: Do I need internet to use the portal?
A: Only if using MongoDB Atlas. Local MongoDB works offline.

Q: Can I access from another computer?
A: Yes, but need to configure network access. See INSTALLATION_GUIDE.md

Q: How long does server take to start?
A: Usually 3-5 seconds

Q: Can I use while server is starting?
A: No, wait for "running on port 3000" message

Q: What if I forget to stop server?
A: It's OK, but better to stop when done to free resources

Q: Can multiple people use at same time?
A: Yes! Multiple browsers can access simultaneously

Q: Will my data be saved?
A: Yes, all data is saved in MongoDB database

Q: What if computer restarts?
A: Just start server again. Data is safe in database.
