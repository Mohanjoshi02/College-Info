Problem: Images still not showing after saving

Solutions:
□ Check filenames are EXACTLY correct (lowercase, no spaces)
□ Check files are in: public\images\ (not in a subfolder)

□ Clear browser cache
□ Check browser console (F12) for errors

Solutions:
□ Check which images are missing
□ Re-save the missing images
□ Verify filenames match exactly
□ Check file isn't corrupted

Problem: Images are too large/slow to load

Solutions:
□ Resize images to 1200x800 pixels
□ Compress images using online tools
□ Keep file size under 500KB each
