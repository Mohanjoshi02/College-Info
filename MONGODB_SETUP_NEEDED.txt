┌─────────────────────────────────────────────────────────────────┐
│ OPTION 1: MongoDB Atlas (EASIEST - Recommended)                │
│ ⏱️  Time: 5 minutes                                             │
│ 💰 Cost: FREE forever                                           │
│ 🌐 Works: From anywhere with internet                           │
└─────────────────────────────────────────────────────────────────┘

STEP-BY-STEP:

1. Open browser and go to:
   https://www.mongodb.com/cloud/atlas/register

2. Sign up (FREE):
   - Use your email or Google account
   - Verify your email

3. Create FREE Cluster:
   - Click "Build a Database"
   - Select "M0 FREE" (512MB - Free forever)
   - Choose AWS as provider
   - Choose region closest to you
   - Click "Create Cluster" (wait 3-5 minutes)

4. Create Database User:
   - Click "Database Access" in left menu
   - Click "Add New Database User"
   - Username: mitadt_admin
   - Password: Create a strong password (SAVE IT!)
   - Database User Privileges: "Atlas admin"
   - Click "Add User"

5. Allow Network Access:
   - Click "Network Access" in left menu
   - Click "Add IP Address"
   - Click "Allow Access from Anywhere"
   - Click "Confirm"

6. Get Connection String:
   - Go back to "Database" (Clusters)
   - Click "Connect" button on your cluster
   - Choose "Connect your application"
   - Select "Node.js" and version "4.1 or later"
   - Copy the connection string
   
   It looks like:
   mongodb+srv://mitadt_admin:<password>@cluster0.xxxxx.mongodb.net/?retryWrites=true&w=majority

7. Update .env file:
   - Open .env file in your project
   - Find this line:
     MONGODB_URI=mongodb://localhost:27017/mitadt_campus
   
   - Replace with (use YOUR actual values):
     MONGODB_URI=mongodb+srv://mitadt_admin:YOUR_PASSWORD@cluster0.xxxxx.mongodb.net/mitadt_campus?retryWrites=true&w=majority
   
   - Replace <password> or YOUR_PASSWORD with your actual password
   - Add /mitadt_campus before the ?
   - SAVE the file

8. Start the server:
   npm start

9. Check console - should see:
   ✅ Connected to MongoDB successfully!
   Default users created successfully!

│ OPTION 2: Install MongoDB Locally                              │
│ ⏱️  Time: 10-15 minutes                                         │
│ 💰 Cost: FREE                                                   │
│ 🌐 Works: Offline (no internet needed)                          │
└─────────────────────────────────────────────────────────────────┘

STEP-BY-STEP:

1. Download MongoDB:
   https://www.mongodb.com/try/download/community
   
   - Select: Windows
   - Package: MSI
   - Click Download

2. Install MongoDB:
   - Run the downloaded .msi file
   - Choose "Complete" installation
   - ✓ Check "Install MongoDB as a Service"
   - ✓ Check "Run service as Network Service user"
   - Service Name: MongoDB
   - Data Directory: C:\Program Files\MongoDB\Server\7.0\data\
   - Log Directory: C:\Program Files\MongoDB\Server\7.0\log\
   - Click Next and Install

3. Wait for installation to complete (5-10 minutes)

4. Verify MongoDB is running:
   - Open Command Prompt
   - Type: mongosh
   - Should connect to MongoDB
   - Type: exit

5. Start your server:
   npm start

6. Check console - should see:
   ✅ Connected to MongoDB successfully!
   Default users created successfully!
